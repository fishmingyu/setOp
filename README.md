## setOp
```bash setOp.sh```
Then run setOp. The first parameter will affect the total number of elements in each list. The second parameter will determine how many lists. The third parameter will determine the value of module.
### major step
1. combine different lists to a global array
2. radix sort
3. merge list and deduplication
4. binary search to find the idx of each element
5. split global array to different lists
